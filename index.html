How to Export from Trackman

1. Open Trackman Baseball software
2. Select your session / pitcher
3. Go to Export → CSV
4. Save the file and upload below

Required columns: PitchType, RelSpeed, SpinRate, HorzBreak, InducedVertBreak, Extension

How to Export from Rapsodo

1. Open Rapsodo Pitching app or web dashboard
2. Select session → Export Data
3. Choose CSV format
4. Upload below

Required columns: pitch_type, velocity, spin_rate, horizontal_break, vertical_break
